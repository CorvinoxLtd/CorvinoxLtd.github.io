Corvinox Limited is a leading software development company located in United Kingdom. We are committed to develop innovative software solutions for leading enterprises in the world, helping them grow their businesses using latest technology solutions.

Job Description

We are looking for an software engineer to join our software development team and help them design, and develop software solutions. Our ideal candidate has a strong technical background in various programming languages and can work with minimum supervision. They like to learn about new technologies and have a strong desire to update their knowledge about the latest industry trends. In their role, the engineer documents the code and collaborates with the team to ensure all codes work per the client's expectations. They participate in all phases of SDLC, including analysis, design, development and testing.

Primary Responsibilities

  • Writing high-quality code and meeting quality standards
  • Improving the software engineering processes and practices
  • Performing unit and integration testing
  • Learning new technologies and finding out about industry trends
  • Document technical requirements for each development project
  • Deliver high-quality products with clean design and fast implementation
  • Optimize applications for maximum speed and scalability
  • Collaborate with the team to organize, implement, and launch new products

Experience and Qualifications

  • Bachelor’s degree in Computer Science or related field (or equivalent experience)
  • Proven experience in designing and building new production systems from scratch
  • Experience working collaboratively with a team to organize, implement, and launch new products
  • Proficiency in at least one programming language, such as Python, or Ruby on Rails
  • 1+ year(s) of recent experience in Ruby on Rails development.
  • Familiarity with test-driven development (TDD)
  • Proficient understanding of code versioning tools, specifically GitHub
  • Experience in designing and developing backend APIs
  • Strong problem-solving skills
  • Excellent communication and teamwork capabilities

Tech Stack

  • Languages: Python or Ruby, JavaScript / Typescript / ActionScript
  • Data: PostgreSQL, DynamoDB, Redis, MongoDB, Solr, Elastic Search
  • Cloud Providers: Amazon Web Services, Google Cloud Platform, Azure
  • Additional Technologies: Kafka, NewRelic, Datadog, Docker, Kubernetes, Git, GitHub, Atlassian
